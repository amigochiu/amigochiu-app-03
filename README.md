# 翻譯蒟蒻 2.0 - 雙向語音翻譯 Web App(Translation Konjac 2.0)

這是一個基於 Web 技術開發的即時語音翻譯應用程式。

## 功能特點
- 支援多國語言即時互譯
- 語音辨識與合成 (Web Speech API)
- 現代化 UI 設計 (玻璃擬態 Glassmorphism)
- 主題切換 (Candy, Summer, Ocean, Forest)

## 如何使用
1. 直接開啟 `index.html` 即可使用。
2. 點擊麥克風圖示開始說話。
3. 點擊中間的交換按鈕可切換翻譯方向。

## 技術堆疊
- HTML5
- CSS3 (Variables, Flexbox)
- JavaScript (Vanilla)
- Lucide Icons
